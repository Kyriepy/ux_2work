WEB

한 컴퓨터에서 만든 문서를 다른 컴퓨터로 옮기기 위해 인터넷을 이용하여 전 세계의 컴퓨터를 거미줄처럼 연결하여 웹 문서를 쉽게 주고 받을 수 있도록 만든 시스템이다.
웹 서버와 웹 클라이언트 기능을 하는 컴퓨터가 있으면, 웹 서버에 존재하는 문서를 인터넷을 통하여 웹 클라이언트로 불러와 사용자에게 보여준다.
웹 서버는 웹 서버 소프트웨어가 필요하고 웹 클라이언트는 웹 브라우저가 필요하다.

1. 인터넷 서비스이다.
2. HTTP 프로토콜을 사용한다.
3. 하이퍼 텍스트 정보 시스템이다.
4. 멀티 미디어이다.

HYPERTEXT

클릭 시 한 문서에서 다른 문서로 바로 넘어갈 수 있는 텍스트이다.
전통적인 텍스트는 모두 순차적으로 읽히는 순서가 정해져 있지만 하이퍼텍스트는 비순차적이다.
기술적 정의는 ‘노드와 링크를 단위로 하는 정보의 구조화 방식’이다. 한 개의 페이지를 노드, 연결점을 링크라고 한다.

사용자가 원하는 정보를 원하는 순서로 읽게 해준다.

WEB 2.0

플랫폼으로서의 웹

기존의 웹은 정보의 공유에만 목적이 있었다면 웹 2.0은 한 페이지에서 여러 가지 서비스를 포함하여 꼭 거쳐가야 하는 공간이 되었다.

집단 지성 공간으로서의 웹

초기의 웹은 웹 서비스 관리자가 웹에 정보를 공유하면 사용자가 읽는 방식이였지만 웹이 발전하며 사용자들도 웹에 정보를 개방하고 공유하여 집단 지성을 실천하게 되었다.
사람들의 생각들과 문화들이 모이다 보니 또하나의 생활 공간으로 발전하였다.
